 GoldTrack

GoldTrack is a beginner-friendly Gold Management System built with HTML, CSS, and JavaScript. It helps manage gold products, customer details, and sales in a simple and attractive dashboard.

 Features
- Add gold products
- Add customers
- Record sales
- Auto calculate total price
- Update stock after sale
- View dashboard stats

 Tech Used
- HTML
- CSS
- JavaScript
 How to Run
1. Download or clone the project
2. Open `index.html` in your browser

Purpose
This project was built for practice and portfolio use to understand basic inventory and sales management logic.
